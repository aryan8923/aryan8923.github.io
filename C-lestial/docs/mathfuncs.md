# ```mathfuncs```

This is a collection of functions written to calculate and view mathematical functions.

## Polynomials
- ```void view_polynomial(int coeff_array_size,prec coeff[coeff_array_size],char variable)```


   This functions prints out the polynomial whose coefficients are given by the array ```coeff[coeff_array_size]```. ```variable``` is simply the variable in the polynomial and is given by ```char``` type.
   Remember that the element at nth index of ```coeff``` array is the coefficient of x^n in the polynomial.
